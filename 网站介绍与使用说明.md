# 法润青苗——未成年人网络安全普法平台

## 网站介绍与使用说明

---

## 一、项目概述

### 项目背景

"法润青苗"是四川大学"法暖万家·守护朝夕"团队打造的未成年人网络安全普法平台。随着互联网深入青少年生活，网络暴力、网络诈骗、消费诱导等问题日益突出。本平台旨在通过生动、易懂的普法内容，帮助未成年人提高网络安全意识，掌握自我保护技能。

### 技术概况

- **部署方式**：Render 云平台（24小时在线，全球可访问）
- **技术栈**：React + TypeScript + Node.js + Express + SQLite
- **核心特性**：全站内容后台可配置，无需修改代码即可更新所有文字、图片、链接

## 二、网站功能一览

网站共包含 **10个页面**，覆盖普法教育、活动展示、资源下载、团队介绍等全方位内容。

### 2.1 首页（/）

首页是信息汇总中心，采用左右分栏布局：

**左侧主内容区：**
- **轮播图 Banner**：3 张自动轮播图，展示平台主题和核心信息
- **三大普法专题入口**：网络暴力、网络诈骗、消费诱导，点击可进入对应专题页
- **新闻动态**：最新 6 条新闻，含封面图、分类标签、日期和摘要

**右侧边栏：**
- **网站公告**：置顶公告优先显示，带红色"置顶"标记
- **数据统计**：展示服务人数、志愿时长、普法课程数等关键指标
- **快速链接**：12377 举报中心、12355 青少年热线、中国普法网等
- **联系我们**：邮箱、电话、地址

### 2.2 网络暴力专题（/cyberbullying）

- **知识科普**：网络暴力定义、常见形式、法律法规（刑法第246条、网络安全法、未成年人保护法）、应对方法
- **案例警示**：真实案例剖析，可展开查看详情
- **求助渠道**：12377、12355、家长/老师、110 报警
- **友善上网倡议**：6 条文明上网行为准则

### 2.3 网络诈骗专题（/fraud）

- **高发类型**：6 种常见诈骗手法（免费皮肤、虚假中奖、冒充熟人等）
- **话术拆解**：4 种诈骗话术逐句揭露真相
- **处置流程**：5 步应急处理指南（停止→告知→保留证据→举报→报警）
- **反诈口诀**：朗朗上口的防骗口诀

### 2.4 消费诱导专题（/consumption）

- **陷阱识别**：5 种常见消费陷阱（算法推荐、直播打赏、游戏充值等）
- **理性消费技巧**：制定预算、延迟决策、货比三家、区分"需要"和"想要"
- **无屏幕乐趣**：5 种替代屏幕的活动建议（阅读、运动、手工、家庭互动、音乐）
- **理性消费公约**：6 条消费行为准则

### 2.5 活动回顾（/activities）

- **5 天活动时间轴**：按天切换，展示每天的活动主题、流程、精彩瞬间
- **每日亮点**：3 条核心收获
- **活动流程**：时间线展示每个时段的活动安排
- **照片墙**：活动精彩瞬间展示

### 2.6 学习资源库（/resources）

- **分类筛选**：课件PPT、科普读物、法律资料、工具卡片、实用模板、设计素材
- **搜索功能**：按标题/描述搜索
- **资源卡片**：每个资源显示封面、类型、大小、预览/下载按钮
- **支持多种格式**：PDF、Word、PPT、视频、图片

### 2.7 项目成果（/achievements）

- **成果数据**：服务人数、志愿时长、课程数、服务社区数
- **成果展示**：调研报告、媒体报道、视频记录、数据统计
- **合作共建**：合作单位/组织列表
- **行动号召**：鼓励更多人参与

### 2.8 关于我们（/about）

- **项目背景**：项目初衷和愿景
- **项目意义**：普法教育、实践育人、长效影响
- **指导老师**：教师/律师/顾问的头像和简介
- **团队成员**：8 名团队成员的头像和分工
- **合作单位**：合作伙伴列表
- **联系方式**：邮箱、地址

### 2.9 新闻详情页（/news/:id）

点击首页新闻动态中的任意一条，跳转到详情页，支持三种内容展示方式：
- **图文内容**：直接显示正文
- **外部链接**：显示"查看原文"按钮，可跳转到公众号文章等
- **文件下载**：显示"下载文件"按钮，可直接下载 PDF/Word 等

### 2.10 公告详情页（/announcements/:id）

与新闻详情页类似，支持三种内容展示方式，置顶公告带红色标记。

### 3.3 常用操作示范

#### 修改轮播图

1. 进入后台 → 内容管理 → 轮播图管理
2. 点击某条记录旁的"编辑"按钮
3. 修改标题、副标题、描述文字
4. 上传图片：点击"图片"字段的上传按钮，或直接粘贴图片链接
5. 点击"保存"

#### 发布新闻

1. 进入后台 → 内容管理 → 新闻动态
2. 点击"添加"按钮
3. 填写标题、摘要、分类、日期
4. 上传封面图
5. 选择"内容类型"：
   - **图文内容**：直接在正文区域写内容
   - **外部链接**：粘贴公众号文章链接
   - **文件下载**：上传 PDF/Word 文件
6. 点击"保存"

#### 发布公告

1. 进入后台 → 内容管理 → 公告管理
2. 点击"添加"按钮
3. 填写标题、日期
4. 选择"是否置顶"：是 = 1，否 = 0
5. 选择内容类型并填写内容
6. 点击"保存"

#### 修改网站基本信息

1. 进入后台 → 站点设置 → 基本设置
2. 修改网站名称、副标题、描述
3. 上传 Logo 图片
4. 修改联系方式
5. 点击"保存所有设置"

#### 上传资源文件

1. 进入后台 → 内容管理 → 资源管理（或文件上传）
2. 点击"添加"按钮
3. 填写标题、选择分类
4. 上传文件或粘贴链接
5. 点击"保存"

### 3.4 配置地图使用

"配置地图"是后台最实用的功能之一：

1. 进入后台 → 配置地图
2. 可以看到所有前端页面对应的后台管理模块
3. 例如：想知道"首页轮播图"在哪改 → 找到"首页"→ 看到"轮播图管理"→ 点击"前往管理"
4. 支持搜索：输入关键词快速定位

### 3.5 网站风格切换（三种版本）

网站提供了 **三套完全不同的视觉风格**，不止配色不同，**布局思路也完全不同**，方便组长根据活动定位选择最合适的版本。

#### 三种风格对比

| 风格 | 名称 | 配色 | 首页布局 | 导航风格 | 适用场景 |
|------|------|------|----------|----------|----------|
| **风格A** | 公益组织风 | 绿色系（emerald） | 左右分栏：主内容 + 侧边栏 | 绿底白字，置顶固定 | 社区普法、NGO项目 |
| **风格B** | 政务教育风 | 蓝红系（navy/blue） | 单栏通栏：大Banner → 区块依次排列 | 深蓝底白字，正式大气 | 政府网站、学校官网 |
| **风格C** | 青春校园风 | 多彩渐变（橙/粉/紫） | 卡片瀑布流：大图大标题，动态数字 | 白色半透明，滚动变色 | 学生社团、青年活动 |

#### 如何切换风格

1. 进入后台 → **站点设置** → 基本设置
2. 滚动到页面底部，找到 **"网站风格"** 区域
3. 点击想要切换的风格卡片（风格A / 风格B / 风格C），选中的卡片会有紫色边框和对勾标记
4. 点击 **"保存所有设置"**
5. 打开网站首页查看效果

#### 如何锁定风格

选定最终风格后，建议锁定以防止误操作：
1. 在"网站风格"区域，打开 **"锁定当前风格"** 开关
2. 点击 **"保存所有设置"**
3. 锁定后，风格切换入口将隐藏，只保留当前版本

如需重新切换风格，点击 **"解锁风格"** 按钮即可。

## 五、常见问题

### Q1：修改后台后，前端没变化？

刷新前端页面即可。修改是即时生效的，不需要重启服务器。

### Q2：上传的图片不显示？

检查图片链接是否正确。两种方式：
- **本地上传**：后台直接上传，自动生成 `/uploads/xxx.jpg` 链接
- **外部链接**：粘贴完整 URL，如 `https://example.com/image.jpg`

### Q3：网站打开很慢？

Render 免费版服务在 15 分钟无访问后会自动休眠，首次访问需要约 50 秒唤醒。唤醒后访问速度正常。

### Q4：如何修改导航菜单？

后台 → 站点设置 → 导航菜单 → 添加/编辑/删除菜单项。

### Q5：如何修改页面底部版权信息？

后台 → 站点设置 → 基本设置 → 修改"版权信息"和"ICP备案号"。

### Q6：数据会丢失吗？

已配置持久化磁盘，数据库和上传文件都会保存，不会丢失。

### Q7：网站可以绑定自己的域名吗？

可以。Render 支持自定义域名，在服务设置中添加即可。

### Q8：忘记密码怎么办？

联系网站管理员，在服务器上重置密码，或重新初始化数据库。
